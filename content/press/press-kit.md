## About LLM Trust

**LLM Trust** is a discovery and evaluation platform for open-source Large Language Models.

In a landscape where new open-source LLMs launch weekly, developers and teams struggle to find the right model for their use case. LLM Trust solves this by providing:

- **Curated model directory** — Every major open-source LLM, benchmarked and categorized
- **Side-by-side comparisons** — Compare models on performance, cost, license, and capabilities
- **Community trust scores** — Real-world feedback from developers using these models in production
- **Updated weekly** — New models added as the ecosystem evolves

### The Problem
The open-source LLM ecosystem is exploding — but fragmented. Developers spend hours researching models across GitHub, Hugging Face, Reddit, and blog posts. There's no single source of truth.

### The Solution
LLM Trust is the **one-stop destination** for discovering, comparing, and choosing open-source LLMs.
