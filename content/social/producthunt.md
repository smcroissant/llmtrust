# Product Hunt Launch Kit — LLM Trust

## Produit

- **Nom:** LLM Trust
- **Tagline (60 chars max):** The npm of LLMs — discover & compare language models
- **URL:** https://www.llmtrust.com

## Description (260 chars max)

```
Discover, compare, and evaluate 100+ LLMs in one place. Side-by-side benchmarks, pricing, context windows, and real-world use cases. Stop guessing which model to use — start building with confidence. Built for developers & ML engineers.
```

## Maker Comment (premier commentaire)

```
Hey Product Hunt! 👋

I'm the founder of LLM Trust, and I'm excited to share what we've been building.

The problem: There are 100+ LLMs available today. Every week brings new models, new benchmarks, new pricing. Developers spend hours researching which model to use — time they should spend building.

The solution: LLM Trust is like npm for language models. Search, compare, and evaluate LLMs side-by-side. See benchmarks, pricing, context windows, latency, and real-world recommendations — all in one place.

What we've built:
🔍 Search & filter 100+ models by capability, price, and use case
📊 Side-by-side benchmark comparisons (MMLU, HumanEval, MT-Bench, etc.)
💰 Real-time pricing across providers
🎯 "Which model should I use?" recommendations by use case
📖 Open source & API models in one unified view

We're starting with English but will expand to more languages soon.

I'd love your feedback — what would make this even more useful for your workflow?

Thanks for checking us out! 🧠
```

## Timing Strategy

### Jour de lancement recommandé

- **Jour:** Mardi ou Mercredi
- **Éviter:** Lundi (weekend backlog), Vendredi (mort), Week-end (moins de trafic)
- **Heure de lancement:** 00:01 AM PST (9:01 AM CET) — le plus tôt possible pour capturer le cycle entier

### Pourquoi mardi/mercredi

- Maximum de trafic PH en milieu de semaine
- Les hunters et early adopters sont les plus actifs
- Assez de temps pour monter dans les trending avant le weekend

## Maximiser les upvotes

### Pré-launch (2-4 semaines avant)

3. **Contacter un Hunter** — Un hunter avec 500+ followers pour lancer le produit
4. **Préparer la communauté** — DM les early supporters, prévenir Discord/Slack

1. **Lancer à 00:01 AM PST**
2. **Partager sur tous les réseaux** dans les 30 premières minutes
