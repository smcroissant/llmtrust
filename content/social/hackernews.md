# Hacker News Strategy — LLM Trust

## Show HN

### Titre (recommandé)

```
Show HN: LLM Trust – Compare 100+ language models side by side
```

### Titres alternatifs

```
Show HN: I built a searchable database of LLMs with benchmarks and pricing
```
```
Show HN: An open tool to help developers pick the right LLM
```

**Règles du titre HN:**
- Commencer par "Show HN:"
- Pas de clickbait, pas de superlatifs
- Descrire ce que c'est, pas ce que ça fait
- Court et factuel

- Filter by context length, license, modality, and more
- Get recommendations by use case (RAG, coding, chat, classification, etc.)

**Tech stack:**
[Your actual stack — be specific. HN loves this.]

**What's next:**
- Quantization support (GGUF, GPTQ, AWQ)

**Questions for the HN community:**
1. Which benchmarks matter most in practice (vs. academic)?
2. What would make this indispensable for your workflow?

Happy to answer any questions.

## Stratégie pour atteindre la front page

### Pré-requis

1. **Compte HN âgé** — Minimum 1 mois, karma > 50
2. **Historique de commentaires** — Avoir commenté sur d'autres posts avant
3. **Pas de spam** — Premier Show HN depuis au moins 2 semaines

### Maximiser les chances

1. **Poster à l'heure exacte** — 8:00 AM EST, pas 7:45 ou 8:30
2. **Être disponible pour répondre** — Les 2 premières heures sont CRITIQUES
3. **Répondre à chaque commentaire** — Rapidement et avec substance
4. **Upvote naturel seulement** — Jamais demander, jamais coordonner
5. **Contenu original** — HN déteste les reposts de contenu déjà publié

### Ce qui marche sur HN

- **Technique depth** — Expliquer comment c'est construit
- **Données réelles** — Share metrics, benchmarks, numbers
- **Humble tone** — "I built this because I needed it"
- **Open source** — Si possible, mentionner le code source
- **Réponses substantielles** — 2-3 paragraphes, pas juste "Thanks!"

### Ce qui tue un post HN

- ❌ Marketing speak
- ❌ Superlatifs ("the best", "revolutionary")
- ❌ Demander des upvotes
- ❌ Réponses courtes ou défensives
- ❌ Lien sans contexte
- ❌ Compte neuf sans historique

## After the post

### Si ça marche (front page)

1. Répondre à TOUS les commentaires pendant 4-8h
2. Monitorer le site pour le traffic spike
3. Prepare-toi au feedback critique (HN est exigeant)
4. Capture les métriques (visiteurs, signups, etc.)

### Si ça marche pas

1. Attendre 2 semaines minimum
2. Améliorer le produit basé sur les retours
3. Reposter avec un angle différent
4. Ou poster un "Show HN" pour une feature spécifique plutôt que le produit entier
