### Activation → L'utilisateur fait-il quelque chose de valuable ?
**Target : 60% activation rate**

User activated = visité 3+ pages ET a comparé 2 modèles

**Optimisations :**
1. **Homepage** → directement une comparison (pas de friction)
2. **"Compare" button** → visible partout
3. **No signup required** → accès immédiat à tout
4. **Quick load** → < 2s page load
5. **Smart suggestions** → "Based on your interest in Llama 3, you might want to compare it with..."

### Retention → Revient-il ?
**Target : 25% week-1 retention**

| Mechanic | Description |

| Newsletter | Weekly digest → re-engagement |
| Model alerts | "New benchmark for [model]" |
| Fresh content | 3 articles/week |
| New models | 5-10 new models/week |
| Community | Discord discussions |

### Referral → Amène-t-il d'autres personnes ?
**Target : k-factor 0.3**

Voir `viral-loops.md` pour les mécanismes détaillés.

## 6. Competitive Moats

### Built Over Time
1. **Data moat** : Largest open LLM benchmark dataset
2. **Community moat** : Active Discord + GitHub community
3. **Content moat** : 36+ SEO-optimized articles
4. **Network moat** : User reviews + UGC
5. **Brand moat** : "LLM Trust" = the LLM comparison platform

### Defensibility
- Programmatic pages are hard to replicate (data + templates)
- Community creates switching costs
- SEO authority compounds over time
- Open source creates ecosystem lock-in

## 7. Risk Mitigation

| Risk | Impact | Mitigation |

| Google algorithm change | HIGH | Diversify channels (don't rely 100% on SEO) |
